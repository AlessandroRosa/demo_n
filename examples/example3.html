                  }
                } );

    _demon.add( { action : "end" } );
    _demon.run();
The pre-function and the post-function are two input properties including a function object: they run before and after the action takes place respectively.

In this example, we create a div on the fly, we move it to the center of the screen and we finally remove it from the DOM.
So we don't need objects to already belong to the DOM: we can dynamically create'em.
