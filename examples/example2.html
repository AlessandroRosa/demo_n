Example 2: simulating human typing
var _demon = new demo_n(true);
    _demon.add( { action : "start" } );

    _demon.add( { ctrl_id : "mytextarea", time : 1, time_unit : "s",
                  action : "clean", set_value : "", showlabel : true, desclabel : "Cleaning the textarea" } );

    var _str = "1st row\n2nd row\n5rd ... oooops" ;
    for( var _i = 0 ; _i < _str.length ; _i++ )
    _demon.add( { ctrl_id : "mytextarea", time : 100, time_unit : "ms",
                  action : "keypress", set_value : _str[_i], showlabel : true, desclabel : "Writing down something" } );

    for( var _i = 0 ; _i < 14 ; _i++ )
    _demon.add( { ctrl_id : "mytextarea", time : 100, time_unit : "ms",
                  action : "keydel", set_value : _str[_i], showlabel : true, desclabel : "...ooops !" } );

    var _str = "3rd row\n4th row" ;
    for( var _i = 0 ; _i < _str.length ; _i++ )
    _demon.add( { ctrl_id : "mytextarea", time : 100, time_unit : "ms",
                  action : "keypress", set_value : _str[_i], showlabel : true, desclabel : "Writing down again" } );

    _demon.add( { action : "end" } );
    _demon.run();
Here's soon why we should think in terms of movies and frames:
first we clean the textarea control, so we can run the demo multiple times, without appending the new text.
Then we take an input string and we add one action per each character therein. After the 'oooops', we delete a number a characters.
Finally we demon-type again one more string.

I choose a time interval of 100 milliseconds, try accelerating it to 50s or slow it down to 200.
